## ✨ المميزات

### 🖥️ لوحة تحكم متقدمة
- **لوحة القيادة الرئيسية** — عرض شامل لحالة النظام بالكامل في الوقت الفعلي
- **مراقبة الأداء** — رسوم بيانية حية لاستخدام CPU والذاكرة والشبكة
- **واجهة RTL كاملة** — دعم اللغة العربية مع تصميم متجاوب

### ⚛️ الحوسبة الكمية
- **إدارة المعالجات الكمية** — مراقبة وتحكم في المعالجات متعددة الكيوبت
- **تشغيل الدوائر الكمية** — محاكاة وتنفيذ خوارزميات كمية
- **تتبع التجارب** — مراقبة تقدم التجارب الكمية في الوقت الفعلي

### 🤖 الذكاء الاصطناعي
- **إدارة نماذج AI** — تدريب ومراقبة نماذج الشبكات العصبية
- **عوامل ذكية** — وكلاء AI للتحليل والمراقبة التلقائية
- **تحليلات متقدمة** — استخدام GPU/CPU ومقاييس الأداء

### 🔒 الأمان المتقدم
- **تشفير كمي** — حماية ما بعد الكم (Post-Quantum Cryptography)
- **مراقبة التهديدات** — كشف وتحليل التهديدات الأمنية
- **تقييم المخاطر** — تحليل شامل للثغرات والتوصيات الأمنية

### 🌐 مراقبة الشبكة
- **خريطة العقد** — عرض حالة جميع عقد الشبكة
- **تحليل الترافيك** — مراقبة النطاق الترددي وزمن الاستجابة
- **إدارة الاتصالات** — بروتوكولات آمنة للاتصال

## 🚀 البدء السريع

### المتطلبات
- **Node.js** 18+ (يُفضل 20 LTS)
- **npm** 9+

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/Top1-Quantum-AI/Top1-Quantum-AI.github.io.git
cd Top1-Quantum-AI.github.io

# تثبيت التبعيات
npm install

# تشغيل الخادم المحلي
npm run dev
```

افتح المتصفح على **http://localhost:3000** 🎉

### أوامر مفيدة

| الأمر | الوصف |
|-------|-------|
| `npm run dev` | تشغيل خادم التطوير |
| `npm run build` | بناء نسخة الإنتاج |
| `npm run preview` | معاينة نسخة الإنتاج |
| `npm run lint` | فحص الكود |
| `npm run format` | تنسيق الكود |
| `npm run type-check` | فحص الأنواع TypeScript |
| `npm test` | تشغيل الاختبارات |
| `npm run server` | تشغيل الخادم الخلفي |

## 🔒 الأمان

- **لا تُضمّن** أي مفاتيح سرية أو بيانات اعتماد في الكود
- استخدم ملفات `.env` لمتغيرات البيئة (راجع `.env.example`)
- التشفير ما بعد الكمي (Post-Quantum) للبيانات الحساسة
- جميع المدخلات مُعقّمة ومُتحقق منها على الخادم
- فحص الثغرات تلقائياً عبر CI/CD (Trivy + Snyk)

## 📖 التوثيق

### متغيرات البيئة

```env
# Frontend (Vite)
VITE_API_URL=http://localhost:3001
VITE_OPENAI_API_KEY=your-key-here

# Backend
PORT=3001
MONGODB_URI=mongodb://localhost:27017/quantum-ai
REDIS_URL=redis://localhost:6379
JWT_SECRET=your-jwt-secret
```

### مسارات API

| الطريقة | المسار | الوصف |
|---------|--------|-------|
| `POST` | `/api/v1/auth/login` | تسجيل الدخول |
| `POST` | `/api/v1/auth/register` | إنشاء حساب |
| `GET` | `/api/v1/quantum/status` | حالة المعالجات الكمية |
| `POST` | `/api/v1/ai/analyze` | تحليل البيانات بالذكاء الاصطناعي |
| `GET` | `/health` | فحص صحة الخادم |
| `GET` | `/ready` | جاهزية الخادم |

## 🤝 المساهمة

نرحب بالمساهمات! يرجى اتباع الخطوات التالية:

1. افتح **Fork** للمستودع
2. أنشئ فرعاً جديداً: `git checkout -b feat/your-feature`
3. اعتمد التغييرات: `git commit -m 'feat: add amazing feature'`
4. ادفع الفرع: `git push origin feat/your-feature`
5. افتح **Pull Request**

### قواعد الالتزام (Commit Convention)

```
feat:     ← ميزة جديدة
fix:      ← إصلاح خطأ
docs:     ← تحديث التوثيق
style:    ← تنسيق (بدون تغيير في المنطق)
refactor: ← إعادة هيكلة
test:     ← إضافة اختبارات
chore:    ← مهام صيانة
```
